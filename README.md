# Anoop Singh & Associates - Redesigned Website

A modern, responsive redesign of the Anoop Singh & Associates website featuring improved UI/UX, better navigation, and contemporary design elements.

## Features

- **Modern Design**: Clean, professional design with a contemporary color scheme
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Subtle animations and transitions for better user experience
- **Portfolio Gallery**: Filterable portfolio section showcasing various projects
- **Contact Form**: Interactive contact form with validation
- **Smooth Scrolling**: Smooth scroll navigation between sections
- **Mobile Menu**: Hamburger menu for mobile devices

## Project Structure

```
dads_website/
├── index.html      # Main HTML file
├── styles.css      # All styling and responsive design
├── script.js       # JavaScript for interactivity
└── README.md       # Project documentation
```

## Getting Started

1. Open `index.html` in a web browser
2. No build process or dependencies required - it's a static website

## Sections

- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **About Section**: Information about Anoop Singh and company statistics
- **Services Section**: Six service cards with icons and descriptions
- **Portfolio Section**: Filterable gallery of completed projects
- **Clients Section**: Showcase of client types
- **Contact Section**: Contact information and inquiry form
- **Footer**: Quick links and company information

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563EB;
    --primary-dark: #1E40AF;
    --secondary-color: #10B981;
    /* ... */
}
```

### Content
Update the content directly in `index.html` - all text is easily editable.

### Images
Replace the placeholder divs with actual images by modifying the HTML and adding image sources.
